NBA v2 launched

The Netherlands Biodiversity API v2 is officially live as of October 25, 2017.

Access the NBA at api.biodiversitydata.nl/v2/
For a complete list of available services, see api.biodiversitydata.nl/v2/metadata/getRestServices
Read the documentation: docs.biodiversitydata.nl

The previous version (v0) is no longer supported and will be discontinued on May 1, 2018. Be aware that NBA v2 is not backward compatible with NBA v0.

For further information please contact the Naturalis development team: nba@naturalis.nl

Naturalis Biodiversity Center. Posted: 25-10-2017
